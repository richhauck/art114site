Where do you start when creating a website? You could obviously type out a new HTML page, or you could keep a template folder around that you copy each time you start a new project. That said, there are some tools out there that can help:

  • HTML5 Boilerplate - an open source and great starting point for creating a website. While it is an opnionated approach, it includes nearly everything you need to get started.
  • Google Web Starter Kit (no longer supported)
  • You can create a new HTML boilerplate page with your text editor using Emmet.
    1. Create a new document in your text editor
    2. Save it with an .html extension
    3. In the file, type html:5 and then hit tab.

Items your starting project should include:

  • Every page should contain the following meta tag (to assist with responsive layouts): {%highlight html %}{% endhighlight %}
  • You should dedicate a folder for vendor, or third-party code. This should be a place that houses third-party code you have downloaded for use in your site, and it should remain untouched by you, as it will make updating easier.
  • A robots.txt file
  • A favicon
  • An Apple Touch Icon
  • An Open Graph image for popular services like Facebook (see: Facebook for Developers Guide)
